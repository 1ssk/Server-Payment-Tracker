## Первые шаги

### ✅ Добавить первый сервер

1. Нажмите кнопку **"Добавить сервер"**
2. Заполните форму:
   - **Название**: например, "My-VPN-Server"
   - **IP адрес**: например, "192.168.1.100"
   - **Провайдер**: например, "DigitalOcean"
   - **Ссылка на хостинг**: https://digitalocean.com (опционально)
   - **Локация**: например, "США, Нью-Йорк"
   - **Статус**: Активный
   - **Стоимость**: например, 1500 (в рублях)
   - **Период оплаты**: Ежемесячно
   - **Дата следующего платежа**: выберите дату
   - **Заметки**: любая дополнительная информация (опционально)
3. Нажмите **"Добавить сервер"**

### ✏️ Редактировать сервер

1. Найдите карточку сервера
2. Нажмите иконку **карандаша** ✏️
3. Измените нужные поля
4. Нажмите **"Сохранить изменения"**

### 🗑️ Удалить сервер

1. Найдите карточку сервера
2. Нажмите иконку **корзины** 🗑️
3. Подтвердите удаление

### 🔍 Найти сервер

1. Введите текст в поле **поиска**
2. Приложение автоматически отфильтрует серверы по:
   - Названию
   - Провайдеру
   - Локации
   - IP адресу

### 🎛️ Фильтровать по статусу

Нажмите на кнопки фильтра:
- **Все** - показать все серверы
- **Активные** - только активные
- **Неактивные** - только неактивные
- **Ожидание** - серверы в процессе настройки

### ⚙️ Настроить email-уведомления

1. Нажмите кнопку **"Настройки"** вверху справа
2. Включите переключатель **"Включить email-уведомления"**
3. Заполните параметры SMTP:
   - **SMTP хост**: smtp.gmail.com
   - **Порт**: 587
   - **Имя пользователя**: your-email@gmail.com
   - **Пароль**: ваш пароль
   - **От кого**: noreply@example.com
   - **Кому**: admin@example.com
4. Нажмите **"Сохранить настройки"**

⚠️ **Важно**: Для реальной отправки email нужен backend-сервер

### 🚪 Выход из системы

Нажмите кнопку **"Выйти"** вверху справа

## 💡 Подсказки

### Цветовые индикаторы платежей

- 🔴 **Красный** - платёж просрочен
- 🟠 **Оранжевый** - осталось ≤3 дней (срочно!)
- 🟡 **Жёлтый** - осталось 4-7 дней (скоро)
- ⚪ **Обычный** - более 7 дней

### Статусы серверов

- 🟢 **Активный** - сервер работает
- ⚫ **Неактивный** - сервер отключен
- 🟡 **Ожидание** - сервер настраивается

### Периоды оплаты

- **Ежемесячно** - годовая стоимость = сумма × 12
- **Ежеквартально** - годовая стоимость = сумма × 4
- **Ежегодно** - годовая стоимость = указанная сумма
